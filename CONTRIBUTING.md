# Contributing

Contributions, corrections, and improvements are welcome!

## How to Contribute

1. Fork the repo
2. Create a branch: `git checkout -b day-X-improvement`
3. Make your changes
4. Open a PR with a clear description

## Ideas for Contributions

- Additional lab scenarios per day
- Sample data files for model training (CSV, PDF)
- PowerFx formula cheat sheet additions
- Video walkthrough links
- Translation to other languages
- Real-world case studies

## Content Guidelines

- Use sentence case in all headings
- Include a "trainer insight" for real-world context per major concept
- All labs must be step-by-step with exact UI navigation
- Mark prerequisites clearly at the start of each lab
- Include expected outcomes for every test step
